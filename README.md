# 📊 Student Attendance Tracker (Web Project)

## 🚀 Project Overview

This is a simple and interactive **Attendance Tracker Website** that helps students calculate their attendance percentage and check exam eligibility.

The system also tells how many more classes a student must attend to reach the required **75% attendance**.

---

## ✨ Features

* 📥 Input student details (Name, Total Days, Present Days)
* 📊 Converts days into classes (1 day = 7 classes)
* 📈 Calculates:

  * Total Classes
  * Present Classes
  * Absent Classes
  * Attendance Percentage
* ✅ Eligibility check (75% criteria)
* ❌ Shows required extra classes if attendance is low
* 📊 Bar Chart Visualization
* 🥧 Pie Chart Visualization

---

## 🧠 How It Works

* Each day contains **7 classes**
* Attendance is calculated based on total classes
* If attendance is below 75%, the system calculates:

  Required classes such that:

  (present + x) / (total + x) ≥ 75%
